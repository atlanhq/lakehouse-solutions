# BigQuery Solutions

Solutions for the Atlan Metadata Lakehouse (MDLH) on BigQuery.

## Available Solutions

| Solution | Description |
|----------|-------------|
| [Gold Layer](./gold-layer/) | Deploy the MDLH Gold Layer — curated, analytics-ready metadata views and tables in BigQuery |
| [External Iceberg Tables](./external-iceberg-tables/) | Create and refresh external Iceberg tables in BigQuery that point to the Atlan Lakehouse — a workaround for querying the MDLH until BigQuery supports native Iceberg REST Catalog federation |
